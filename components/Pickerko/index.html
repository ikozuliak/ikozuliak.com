Pickerko

A lightweight, easily customizable, datepicker plugin, written in MVC style with vanilla JS. Why? Why not!?

Basic usage

<div class="dropdown" >
    <input type="text" id="pickerko"/>
</div>

new Pickerko(document.getElementById('pickerko'))

Using as a component

<input type="text" id="targetInput"/>
<div id="pickerko"></div>

var container = document.getElementById('pickerko');
var pickerko = new Pickerko(container, {
    targetInput:'targetInput',
});

Daterange picker, even in one window

Left click for start of period, right for the end. 

<input type="text" id="targetInput"/>
<div id="pickerko"></div>

var container = document.getElementById('pickerko');
var pickerko = new Pickerko(container, {
    startDate : '17/06/2013',
    endDate : '26/06/2013',
    targetInput:'targetInput3',
    rangePicker:true
});

Left-left click mode

The same as above, but in another style, first left click to choose period start and second one for end of period. 

<input type="text" id="targetInput"/>
<div id="pickerko"></div>

var container = document.getElementById('pickerko');
var pickerko = new Pickerko(container, {
    startDate : '17/06/2013',
    endDate : '26/06/2013',
    targetInput:'targetInput3',
    rangePicker:true,
    rangeClickMode: 'left-left'
});

API and callbacks



pickerko5.setRange('17/02/2013', '22/02/2013')

alert(pickerko5.getRange())

new Pickerko(document.getElementById('pickerko'), {
        onUpdate:function (newDate) {
            alert('Date changed to ' + newDate);

});
